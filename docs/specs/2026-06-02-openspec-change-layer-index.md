# OpenSpec Change 分层索引

> 快照日期：2026-06-02。当前 active changes 共 89 个。

本索引用于给 AI 产品复杂度分层。它不是归档清单，也不要求删除现有能力。后续新增 change 时，先判断它属于哪一层，再决定是否能进入学生主链路。
