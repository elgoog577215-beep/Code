#### 题目与代码理解
这道题要求实现动态连通性，核心是维护若干个集合，支持“合并集合”与“查询是否在同一集合”两种操作。你选择了并查集（DSU）这一数据结构，整体框架包括初始化、查找根节点 `find`、以及主循环中的操作分发，方向是完全正确的。代码能够编译通过，说明语法基础扎实。但目前判题结果为 `WRONG_ANSWER`，意味着程序逻辑在特定测试点上输出了错误结果，我们需要深入检查并查集的核心逻辑。

   标准库命中：HIT，相关坐标：条件分支遗漏
   证据：来自代码第 8 行 `if(find(a)==find(b)) p[find(a)] = find(b);` 以及题目要求“支持加边”。
   给学生的解释与建议：

   标准库命中：HIT，相关坐标：复杂度过高
   证据：来自代码第 3 行 `int find(int x){ return p[x]==x?x:find(p[x]); }`。

   标准的并查集查找通常包含“路径压缩”，即在递归返回时将沿途节点的父节点直接指向根节点。建议在上一步修复逻辑错误后，务必补充这一优化，以保证算法效率。
